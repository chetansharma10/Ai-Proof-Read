# AI Writing Tool

## Overview
The AI Writing Tool is a desktop application that helps users proofread and improve their writing. It uses a powerful AI model to detect and correct grammar errors, typos, and other issues in the text. The tool also provides suggestions for improving the clarity and readability of the writing.

## Features
- **Proofreading**: The AI Writing Tool uses a powerful AI model to detect and correct grammar errors, typos, and other issues in the text.
- **Suggestions**: The tool provides suggestions for improving the clarity and readability of the writing.
- **Integration**: The tool integrates with popular writing applications, making it easy to use in your daily workflow.

## Installation
To install the AI Writing Tool, follow these steps:

1. Clone the repository.
2. Install the necessary dependencies by running `npm install`.
3. Start the application using `npm run watch-ts`.
4. And then start the application using `npm run start`.
5. To compile ts files run `npm run build`.
6. Build the electron in executable form using `npm run electron-build`.

## Usage
To use the AI Writing Tool, follow these steps:

1. Press command + shift + R to read the clipboard and open the popover with the selected text.
2. It will read the text and improve it using the AI model.
3. The tool will provide suggestions for improving the clarity and readability of the writing.
